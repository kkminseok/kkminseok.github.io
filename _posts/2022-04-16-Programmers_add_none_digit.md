## 3. 생각한 것들(문제 접근 방법)

-----  

## 4. 접근 방법을 적용한 코드

```python
def solution(numbers):
    result = 1+2+3+4+5+6+7+8+9
    ans = sum(numbers)
    return result - ans
```
